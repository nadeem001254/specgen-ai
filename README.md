## 📋 What is specgen-ai?

specgen-ai is a Windows application that helps you create tests automatically by analyzing screen recordings. You only need to record your actions on the screen. The software then understands what you did and creates tests that can check if your apps work correctly. This tool uses artificial intelligence to make test creation faster and easier.

You do not need programming knowledge to use specgen-ai. It works by watching what you do and writing tests for you. This can save time for developers and testers who want to verify their software without manual coding.
